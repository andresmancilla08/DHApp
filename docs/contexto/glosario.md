# Glosario y Entidades

## Términos del dominio (Daggerheart)
- **Hope / Fear:** recursos meta del juego. **Dominios (Domains):** fuentes de cartas/habilidades.
- **Ancestry / Community:** ascendencia y comunidad del personaje. **Cartas:** ~189 en la wiki/contenido.
- **CoreBook:** libro oficial; fuente canónica.

## Entidades principales
- **Personaje / Character** — hoja editable; se edita el personaje solo en nivel 1 (regla de producto). Subida de nivel con flujo propio.
- **Usuario** — autenticado por user + PIN.

## Siglas / nombres internos
- **Grimheart = DHApp** (nombre de producto). Prod: grimheart.co.
- **DPCGL** — licencia bajo la que se usa el contenido (proyecto no oficial).
